# Module 1 | Class 3 Assignment: AI in Uzbekistan — Find It

**Related slides:** Class 3 — ITES Applications Deep-Dive (Recommendations, Fraud Detection, Chatbots, Churn)
**Estimated time:** 45 minutes

## Objective
Research real AI/ML applications currently in use in Uzbekistan and identify the ML techniques behind them.

## What You Need
- Tool: Google Search + Google Docs (or any word processor)
- No code required
- Reference: Class 3 notes on ITES case studies (recommendations, fraud detection, chatbots, churn prediction)

## Tasks

### Task 1: Find 3 Real AI/ML Applications in Uzbekistan
**What to do:** Research and identify 3 real AI/ML applications currently used in Uzbekistan across different sectors.

**How to do it:**

- Step 1: Open Google and search for AI/ML usage in Uzbek companies. Good search terms: "Uzum AI", "Payme machine learning", "Uztelecom AI", "Humans.uz AI", "MyTaxi algorithm", "e-government Uzbekistan AI".
- Step 2: Check company websites, press releases, LinkedIn posts from employees, and tech news sites (spot.uz, kun.uz, daryo.uz).
- Step 3: For each application, identify: what the product/feature does (not the company mission statement), what type of ML it likely uses, and what data it needs.
- Step 4: If you cannot find confirmed technical details, make an educated guess based on what you learned in class and explain your reasoning.

**Tool:** Google Search for research, Google Docs for writing

**Help if stuck:** Microsoft ML-For-Beginners, Lesson 1 (real-world ML applications overview): https://github.com/microsoft/ML-For-Beginners

**Slide reference:** This was covered in Class 3, Slides: "How Netflix, YouTube, and Uzum Know What You Want" and "Catching Criminals with Machine Learning"

### Task 2: Summary Table
**What to do:** Present your findings in a structured table.

**How to do it:**

- Step 1: Create a table in your document with 4 columns: Application | Sector | ML Type | Data Used.
- Step 2: Fill one row per application. Be specific — "NLP chatbot for billing questions" is good, "uses AI" is not.
- Step 3: In the ML Type column, specify: supervised (classification/regression), unsupervised (clustering), reinforcement learning, or NLP. If it is a combination, say so.

**Tool:** Google Docs — Insert > Table (4 columns, 4 rows including header)

**Slide reference:** This was covered in Class 3, Slide: "ITES Applications — Four Case Studies"

### Task 3: Explanation Paragraphs
**What to do:** Write 1 paragraph (3-5 sentences) per application explaining your reasoning for the ML type classification.

**How to do it:**

- Step 1: For each application, explain the logic: "This is likely supervised classification because the system needs labeled examples of [X] to predict [Y]."
- Step 2: Reference what data would be needed and how the model would be trained.
- Step 3: If you are guessing, say so honestly and explain your reasoning. Good reasoning on a guess is worth more than a confident wrong answer.

**Tool:** Same Google Doc

**Help if stuck:** Google Introduction to AI (Coursera, free to audit) — covers how AI is applied across industries: https://www.coursera.org/learn/google-introduction-to-ai

**Slide reference:** This was covered in Class 3, Slides: "Chatbots — Automating Customer Support" and "Churn Prediction — Keeping Customers Before They Leave"

## Tips
- Avoid vague descriptions. "Uses AI for customer service" is weak. "Uses an NLP chatbot trained on customer support logs to answer billing questions" is strong.
- Look across sectors: telecom, banking, e-commerce, government services, agriculture, transportation.
- Good starting points: Uzum, Payme, Click, Humans.uz, MyTaxi, government digital services.

## Submission
- Written document, 1-2 pages (PDF or Google Doc link).
- Must include: the summary table (4 columns) and 3 explanation paragraphs.
- Due before the next class.
