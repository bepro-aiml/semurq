### Task 1: Vectors and Distances
**What to do:** Pick 3 products from the Superstore dataset, represent each as a 3D feature vector, and compute Euclidean distances between all pairs.

**How to do it:**

- Step 1: Load the data and pick 3 products:
  ```python
  import pandas as pd
  import numpy as np

  df = pd.read_csv('superstore_cleaned.csv')
  # Pick 3 products (use the first 3 rows or any 3 you like)
  products = df[['Sales', 'Quantity', 'Discount']].head(3)
  print(products)
  ```

- Step 2: Extract each product as a NumPy array:
  ```python
  p1 = products.iloc[0].values
  p2 = products.iloc[1].values
  p3 = products.iloc[2].values
  print(f"Product 1: {p1}")
  print(f"Product 2: {p2}")
  print(f"Product 3: {p3}")
  ```

- Step 3: Compute Euclidean distance manually first. For vectors a and b: distance = sqrt((a1-b1)^2 + (a2-b2)^2 + (a3-b3)^2). Write this calculation in a text cell or comment.
- Step 4: Verify with NumPy:
  ```python
  d12 = np.linalg.norm(p1 - p2)
  d13 = np.linalg.norm(p1 - p3)
  d23 = np.linalg.norm(p2 - p3)
  print(f"Distance P1-P2: {d12:.4f}")
  print(f"Distance P1-P3: {d13:.4f}")
  print(f"Distance P2-P3: {d23:.4f}")
  ```

- Step 5: In a text cell, answer: Which two products are most "similar" (smallest distance)? Does this match your intuition from looking at the raw numbers?

**Tool:** `np.linalg.norm(a - b)` — computes Euclidean distance between two vectors

### Task 2: Linear Combinations (Weighted Sum)
**What to do:** Define a weighted scoring formula and apply it to 5 products, then verify with NumPy.

**How to do it:**

- Step 1: Define the formula: `score = 0.5 * Sales + 0.3 * Quantity - 0.2 * Discount`
- Step 2: Select 5 products:
  ```python
  five_products = df[['Sales', 'Quantity', 'Discount']].head(5)
  print(five_products)
  ```

- Step 3: Compute manually for each row. In a text cell, show the calculation for at least one product: "Product 1: 0.5 * 261.96 + 0.3 * 2 - 0.2 * 0.0 = ..."
- Step 4: Verify using NumPy:
  ```python
  weights = np.array([0.5, 0.3, -0.2])
  scores = five_products.values @ weights  # matrix-vector multiplication
  # Or equivalently: np.dot(five_products.values, weights)
  for i, score in enumerate(scores):
      print(f"Product {i+1}: score = {score:.2f}")
  ```

- Step 5: In a text cell, answer: What does this "score" represent conceptually? Could you use it to rank products? What would changing the weights do?

**Tool:** `np.dot()` or `@` operator — dot product / matrix-vector multiplication

**Help if stuck:** 3Blue1Brown — Essence of Linear Algebra, Chapter 1: https://www.youtube.com/watch?v=fNk_zzaMoSs

**Slide reference:** This was covered in Class 4, Slides: "Dot Product — Measuring Similarity" and "Matrices — Tables of Numbers"

## Tips
- Think of each product as a point in 3D space. Distance tells you how far apart two points are. In ML, "similar" items have small distances between their feature vectors.
- The linear combination is literally what a linear model does — it assigns weights to features and adds them up. You just did a forward pass of a linear model.
- The `@` operator in NumPy is the same as `np.dot()` — it computes matrix multiplication. Use whichever feels more readable.
