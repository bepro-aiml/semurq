### Task 3: Write a Function
**What to do:** Write a function called `calculate_average` that takes a list of numbers and returns the average. Test it with at least 2 different lists.

**How to do it:**

- Step 1: In a new code cell, define the function:
  ```python
  def calculate_average(numbers):
      total = sum(numbers)
      count = len(numbers)
      return total / count
  ```

- Step 2: Test it with two different lists:
  ```python
  test1 = [80, 90, 70, 85, 95]
  test2 = [55, 62, 48, 71]
  print("Average 1:", calculate_average(test1))
  print("Average 2:", calculate_average(test2))
  ```

- Step 3: Verify the output makes sense (mentally check one of the averages).

**Tool:** `def`, `sum()`, `len()` — Python built-ins

**Help if stuck:** Kaggle Python Course, Lesson 2 "Functions and Getting Help": https://www.kaggle.com/learn/python

### Task 4: Loops with enumerate()
**What to do:** Write a `for` loop that iterates over a list of at least 5 items and prints each item with its index.

**How to do it:**

- Step 1: Create a list of at least 5 items (e.g., city names, student names, or course topics).
- Step 2: Use `enumerate()` to get both the index and the value:
  ```python
  cities = ["Tashkent", "Samarkand", "Bukhara", "Namangan", "Fergana"]
  for index, city in enumerate(cities):
      print(f"Index {index}: {city}")
  ```

- Step 3: Run the cell and verify the output shows indices 0-4 with corresponding items.

**Tool:** `enumerate()`, `for` loop, f-strings

### Task 5: Combine Everything
**What to do:** Create a list of 10 random exam scores (integers 0-100), calculate the average, and print whether the class passed or failed (threshold: 60).

**How to do it:**

- Step 1: Create a list of 10 scores (make them up or use `random`):
  ```python
  import random
  exam_scores = [random.randint(0, 100) for _ in range(10)]
  print("Scores:", exam_scores)
  ```

- Step 2: Use your `calculate_average` function:
  ```python
  avg = calculate_average(exam_scores)
  print(f"Class average: {avg:.1f}")
  ```

- Step 3: Add a pass/fail check:
  ```python
  if avg >= 60:
      print("Result: PASS")
  else:
      print("Result: FAIL")
  ```

**Tool:** `random.randint()`, your `calculate_average()` function, `if/else`
