Module 2: Programming and Mathematical Foundations — Resources

Curated learning materials organized by content section. Items marked Required are mandatory; everything else is recommended.

Python Foundations (Section 1)

Videos: - freeCodeCamp — Python for Data Science (~12h, assign in chunks) [Beginner] https://www.freecodecamp.org/news/python-data-science-course-matplotlib-pandas-numpy/ Covers NumPy, Pandas, Matplotlib end-to-end. Split across weeks 1-2.

Readings: - Python Data Science Handbook by Jake VanderPlas — Ch 2 (NumPy), Ch 3 (Pandas), Ch 4 (Matplotlib) [Beginner] https://jakevdp.github.io/PythonDataScienceHandbook/ Free online. Good reference to pair with video content.

  • Kaggle Python Micro-Course (~5h, interactive, in-browser) [Beginner] Required https://www.kaggle.com/learn/python Hands-on exercises with instant feedback. No local setup needed.

  • Kaggle Pandas Micro-Course (~4h, interactive) [Beginner] Required https://www.kaggle.com/learn/pandas Covers indexing, grouping, merging — the operations students will use constantly.

Mathematical Foundations (Section 2)

Videos: - 3Blue1Brown — Essence of Linear Algebra, Chapters 1-4 (~1.5h) [Beginner] Required https://www.3blue1brown.com/topics/linear-algebra Vectors, linear transformations, matrix multiplication, determinants. Visual intuition over formulas.

  • Khan Academy — Statistics & Probability [Beginner] https://www.khanacademy.org/math/statistics-probability Assign selectively: Displaying data, Probability basics, Random variables.

  • StatQuest — Histograms, Distributions, P-values, Hypothesis testing (~2h total) [Beginner] https://statquest.org/video-index/ Pick per topic as needed. Josh Starmer explains stats without jargon.

Data Visualization and EDA (Section 3)

Readings: - Kaggle Data Visualization Micro-Course (~4h, Seaborn-based) [Beginner] https://www.kaggle.com/learn/data-visualization Line charts, scatter plots, distributions, custom styles. Practical exercises included.

Google Cloud Tools (Section 4)

Google Skills Boost: - Google Cloud Big Data and Machine Learning Fundamentals [Beginner] Required https://www.cloudskillsboost.google/course_templates/3 Covers BigQuery, data pipelines, GCP ML ecosystem overview.

Datasets

  • Superstore Sales dataset [Beginner] https://www.kaggle.com/datasets/vivek468/superstore-dataset-final Used for Activities 1-4 (EDA, visualization, segmentation).

Tools

  • Google Colab — https://colab.research.google.com Primary lab environment. Free GPU, no setup.

  • Kaggle Notebooks — https://www.kaggle.com/code Alternative environment. Built-in datasets, useful for micro-course exercises.
